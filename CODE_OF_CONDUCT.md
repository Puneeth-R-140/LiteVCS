# Code of Conduct

## Our Pledge

We as members, contributors, and maintainers pledge to make participation in LiteVCS a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

### Positive Behavior ‚úÖ

- **Be respectful** - Value different viewpoints and experiences
- **Be constructive** - Provide helpful feedback and suggestions
- **Be collaborative** - Work together towards common goals
- **Be patient** - Remember everyone is learning
- **Be welcoming** - Make newcomers feel included
- **Give credit** - Acknowledge others' contributions

### Unacceptable Behavior ‚ùå

- Harassment, trolling, or insulting comments
- Personal or political attacks
- Publishing others' private information
- Spam or off-topic content
- Any conduct that could reasonably be considered inappropriate

## Educational Focus

LiteVCS is a **learning project**. We especially encourage:

- **Questions** - No question is too basic
- **Learning in public** - Share your journey
- **Mistakes** - They're part of learning
- **Experimentation** - Try new approaches
- **Teaching** - Help others learn

## Responsibilities

### Maintainers

- Set a positive example
- Provide constructive feedback
- Be responsive to issues and PRs
- Enforce the code of conduct fairly

### Contributors

- Follow the code of conduct
- Be respectful in all interactions
- Accept constructive criticism gracefully
- Focus on what's best for the community

## Scope

This Code of Conduct applies to:

- GitHub repository (issues, PRs, discussions)
- Code comments and documentation
- Any official LiteVCS communication channels
- Representing the project in public spaces

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers. All complaints will be reviewed and investigated promptly and fairly.

### Consequences

Violations may result in:

1. **Warning** - First offense, minor violation
2. **Temporary ban** - Repeated or serious violations
3. **Permanent ban** - Severe or continued violations

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

## Questions?

If you have questions about this Code of Conduct, please open a discussion or contact the maintainers.
